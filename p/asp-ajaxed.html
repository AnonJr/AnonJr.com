ASP-Ajaxed

Michal Gabrukiewicz and a number of other contributors created an awesome library for those using ASP Classic. Michal passed away, but the site remained active.

In mid-2013 the official site disappeared. I was able to contact one of the other programmers that worked with Michal and received permission to take ownership of the project. I've kept the official site and the Google Code Project going since then.

With Google looking more and more like they're going to "retire" Google Code, I plan on moving the code to GitHub. There's also a plan to re-design for the official site, but no ETA as work, school, and life in general have curtailed some of my free time.
